## О проекте

[Coolify](https://coolify.io) — self-hosted PaaS с открытым исходным кодом: альтернатива Heroku, Netlify и Vercel, которую вы разворачиваете на собственном сервере.

Этот проект — русский перевод официальной документации с удобным порталом для чтения прямо в браузере.

## Возможности портала

- 📚 Полный перевод документации (~250 страниц)
- 🔍 Полнотекстовый поиск — ищет как по названиям разделов, так и по содержимому страниц
- 🌙 Тёмная и светлая тема
- 📱 Адаптивный дизайн (мобильные устройства)
- ⚡ Работает без сервера — чистый HTML + JS + Markdown

## Локальный запуск

Для корректной работы поиска нужен HTTP-сервер (из-за `fetch` запросов):

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

Затем открой `http://localhost:8080`
