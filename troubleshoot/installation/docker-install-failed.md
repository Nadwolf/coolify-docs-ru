# Ошибка установки Docker

Если скрипт установки Coolify завершается ошибкой на этапе **“Installing Docker”**, это чаще всего вызвано тем, что на сервере установлена **не-LTS версия операционной системы** (особенно часто это встречается на системах **Ubuntu**).

## Симптомы
Скрипт установки Coolify выдает ошибку вида:
```sh
ERROR: '27.0' not found amongst apt-cache madison results
```

## Решение
- Установите Docker вручную
   - Следуйте официальному [руководству по установке Docker](https://docs.docker.com/engine/install/#server), чтобы вручную установить Docker версии **24+**.

ИЛИ
   
- Используйте LTS-версию ОС
   - Перейдите на версию ОС с длительной поддержкой (**LTS**), например **Ubuntu 22.04 LTS** или **Debian 12**.
