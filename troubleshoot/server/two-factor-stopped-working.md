# 2FA перестала работать

Обычно это проблема синхронизации времени.

## Диагностика
- Проверьте время на вашем сервере командой `date` — если оно неверное, его нужно синхронизировать.
- Проверьте настройки NTP в файле `/etc/ntp.conf`.
- Проверьте статус службы `systemctl status systemd-timesyncd.service`, если ваша ОС использует systemd для синхронизации времени.
- Проверьте правила брандмауэра (`ufw`, `iptables`) на наличие блокировок портов для синхронизации времени (`123/udp`, `123/tcp`).

## Решение
- Если ваша ОС использует systemd, вы можете включить синхронизацию времени командой: `sudo timedatectl set-ntp true`.
- Если ваша ОС не использует systemd, вы можете синхронизировать время командой: `sudo ntpdate ntp.ubuntu.com`.
