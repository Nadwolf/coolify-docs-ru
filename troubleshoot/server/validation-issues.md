# Ошибки при валидации сервера

Вы не можете подтвердить свой сервер из-за ошибки валидации.

## Симптомы
- Во время подтверждения (валидации) вы получаете ошибку `error in libcrypto`.

## Решение
Проверьте правильность добавленного в Coolify приватного ключа. Скорее всего, в нем не хватает обязательных строк: `-----BEGIN OPENSSH PRIVATE KEY-----` и `-----END OPENSSH PRIVATE KEY-----`.
