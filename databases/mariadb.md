## Что такое MariaDB

MariaDB — это ответвление (fork) MySQL с открытым исходным кодом, созданное с целью оставаться свободным. Она призвана быть прямой заменой (drop-in replacement) MySQL с расширенными функциями и производительностью. MariaDB сохраняет высокую совместимость с MySQL, предлагая при этом дополнительные движки хранения (storage engines), улучшения производительности и новые возможности.

Разработка MariaDB была начата ведущими участниками проекта MySQL; она представляет собой надежное и масштабируемое решение для баз данных, подходящее для самого широкого спектра приложений.
