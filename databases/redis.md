# Redis

![Redis](https://coolify.io/docs/images/database-logos/redis.svg)

## Что такое Redis

Redis — это хранилище данных в оперативной памяти, используемое миллионами разработчиков как кэш, векторная база данных, документо-ориентированная база данных, движок для потоков и брокер сообщений. Redis имеет встроенную репликацию и различные уровни персистентности на диске. Он поддерживает сложные типы данных (например, строки, хэши, списки, наборы, отсортированные наборы и JSON) с атомарными операциями над этими типами данных.
