# Терминал (Terminal)

**Coolify** предоставляет мощный **встроенный веб-терминал**, который обеспечивает бесшовный доступ ко всем вашим ресурсам прямо из браузера. Эта функция упрощает управление вашей инфраструктурой и взаимодействие с ней.

Ключевые особенности:

1. **Универсальный доступ**: доступен для всех ресурсов, **управляемых через Coolify**. Для любых контейнеров или серверов.
2. **Поддержка цвета, текстовых редакторов и прочего**: работа терминала основана на [xterm.js](https://xtermjs.org/), что обеспечивает полноценный терминальный опыт.
3. **Отсутствие необходимости во внешних SSH-клиентах**: встроенный терминал избавляет от необходимости использовать сторонние SSH-клиенты или настраивать сложные соединения, оптимизируя ваш рабочий процесс.

#### Это безопасно?

Да. Команды терминала выполняются внутри среды Coolify через защищенное SSH-соединение, что гарантирует безопасность передачи данных.

#### Как получить доступ?

Вы можете открыть терминал через боковую панель.

#### Как это работает **на самом деле**?

Терминал — это веб-интерфейс, позволяющий взаимодействовать с ресурсами через эмулятор терминала. Связь между браузером и терминалом осуществляется через соединение по протоколу Websocket.

После этого создается новый процесс на сервере терминала (внутри контейнера `realtime` в Coolify), который подключается по SSH к основному инстансу Coolify.

Соединение не является прямым: сначала оно проходит аутентификацию через Websocket, затем устанавливается связь внутри основного инстанса Coolify (хост-сервера) для проверки прав доступа к ресурсам.

Затем процесс внутри основного контейнера Coolify устанавливает новое SSH-соединение с целевым ресурсом (контейнером или сервером). Схема ниже наглядно объясняет этот процесс:
