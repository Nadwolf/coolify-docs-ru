# Создание пользователя Root через переменные окружения

Создание пользователя root во время установки необязательно, но рекомендуется, так как это предотвращает доступ к странице регистрации посторонним лицам.

## Требования к валидации

Для учетных данных пользователя root в продакшн-окружении должны быть соблюдены следующие требования.

### Email
- Должен быть корректным адресом электронной почты.
- Должен иметь действующую DNS-запись.
- Максимальная длина: 255 символов.

### Имя пользователя (Username)
- Минимальная длина: 3 символа.
- Максимальная длина: 255 символов.
- Может содержать только буквы, цифры, пробелы, подчеркивания и дефисы.

### Пароль
- Минимальная длина: 8 символов.
- Должен содержать заглавные и строчные буквы.
- Должен содержать хотя бы одну цифру.
- Должен содержать хотя бы один специальный символ.
- Не должен быть часто используемым или скомпрометированным.

## Автоматизированный метод установки

1. **Подготовьте учетные данные**
   Создайте учетные данные для пользователя root в соответствии с требованиями выше.

2. **Запустите команду установки**
   Выполните автоматизированный скрипт установки с вашими учетными данными:

   ```bash
   env ROOT_USERNAME=RootUser ROOT_USER_EMAIL=example@example.com ROOT_USER_PASSWORD=Password bash -c 'curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash'
   ```
   > Посмотреть [исходный код скриптов](https://github.com/coollabsio/coolify/blob/main/scripts/install.sh)

::: info
  Скрипт установки должен запускаться от имени `root`. Если вы не под `root`, используйте `sudo` для повышения привилегий.
  ```bash
  sudo -E env ROOT_USERNAME=RootUser ROOT_USER_EMAIL=example@example.com ROOT_USER_PASSWORD=Password bash -c 'curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash'
  ```
:::

::: warning
  Если какое-либо значение переменной содержит пробел, заключите его в двойные кавычки, например `ROOT_USERNAME="Root User"`.
:::

3. **Завершите настройку**
   После настройки учетных данных root продолжите выполнение [шагов установки](/get-started/installation#quick-installation-recommended) для завершения настройки Coolify.

## Ручной метод установки

1. **Подготовьте учетные данные**
   Создайте учетные данные для пользователя root согласно требованиям выше.

2. **Настройте переменные окружения**
   Отредактируйте файл переменных окружения:

   ```bash
   nano /data/coolify/source/.env
   ```

   Добавьте следующие переменные с вашими данными:
   ```bash
   ROOT_USERNAME=RootUser
   ROOT_USER_EMAIL=example@example.com
   ROOT_USER_PASSWORD=Password
   ```

::: warning
  Если в значениях переменных есть пробелы, используйте двойные кавычки, например `ROOT_USERNAME="Root User"`.
:::

3. **Завершите настройку**
   После настройки учетных данных root продолжите выполнение [шагов установки](/get-started/installation#quick-installation-recommended) для завершения настройки Coolify.
