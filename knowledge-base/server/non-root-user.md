# Пользователь без прав root (Non-root user)
Вы можете использовать сервер с пользователем без прав root для управления вашими ресурсами вместо пользователя root.

Для корректной работы необходимо правильно настроить сервер.

::: danger ВНИМАНИЕ!
**Это экспериментальная функция.**
:::

## Требования

- У пользователя без прав root должен быть добавлен SSH-ключ на сервер.
- Пользователь должен обладать правами sudo.

## Права Sudo

Вам необходимо добавить следующую строку в файл `/etc/sudoers`:

```bash
# Разрешить пользователю (замените на имя вашего пользователя) выполнять команды от имени root без пароля
имя-вашего-пользователя ALL=(ALL) NOPASSWD: ALL
```

Это позволит пользователю выполнять любые команды от имени root без ввода пароля.
Примечание: не забудьте заменить `имя-вашего-пользователя` на реальный логин.

::: warning Осторожно
  Это не самый безопасный способ настройки пользователя без прав root, но в будущем мы планируем улучшить этот механизм, добавив более гранулярные разрешения для конкретных исполняемых файлов.
:::
