# Обзор Sentinel

[Sentinel](https://github.com/coollabsio/sentinel) — это легковесный контейнер с открытым исходным кодом, который обеспечивает:
- API для системы Linux.
- Мониторинг ресурсов сервера (на данный момент — использование CPU и RAM).
- Мониторинг ресурсов контейнеров (на данный момент — использование CPU и RAM).

## Настройка

### Включение Sentinel

1. Перейдите в `Servers` > `<ВАШ_СЕРВЕР>` > `Configurations` > `General`.
2. Найдите раздел `Sentinel`.
3. Включите опцию `Enable Sentinel`.
4. Подождите немного, пока контейнер скачается и запустится.

### Включение метрик (Опционально)

В том же разделе вы можете включить сбор метрик. После активации вам станут доступны:
- Загрузка процессора (CPU usage).
- Потребление оперативной памяти (RAM usage).

::: info Примечание
Сбор метрик на данный момент НЕ доступен для Docker Compose и деплоев на основе шаблонов сервисов (Service Templates).
:::

## Просмотр метрик

### Метрики сервера
Общие метрики сервера доступны по пути:

`Servers` > `<ВАШ_СЕРВЕР>` > `Configurations` > `Metrics`

### Метрики контейнеров
Чтобы посмотреть метрики конкретного контейнера:
1. Перейдите к нужному ресурсу.
2. Откройте вкладку `Configurations`.
3. Выберите подвкладку `Metrics`.
