# Доступ к терминалу (Terminal Access)
Функция **Доступ к терминалу** позволяет включать или отключать доступ к терминалу для вашего сервера и всех запущенных на нем контейнеров прямо из дашборда Coolify.

Эта функция обеспечивает централизованный контроль над доступом к терминалу, повышая безопасность и позволяя администраторам быстро ограничивать доступ при необходимости.

::: warning ВАЖНО
Эта функция появилась в версии **v4.0.0-beta.419**. Чтобы следовать этому руководству, вы **должны** использовать версию v4.0.0-beta.419 или выше.
:::

## Как использовать доступ к терминалу
<ZoomableImage src="/docs/images/knowledge-base/servers/terminal-access/1.webp" alt="Настройки терминального доступа во вкладке безопасности сервера" />

1. Перейдите в раздел **Servers** в боковой панели.
2. Выберите нужный сервер из списка.
3. Откройте вкладку **Security**.
4. Найдите раздел **Terminal Access**.
5. Нажмите кнопку **Disable Terminal**, чтобы отключить терминал (или **Enable Terminal**, если он уже отключен).

::: warning Важно
Отключение доступа к терминалу затрагивает все терминалы на сервере и в его контейнерах. Доступ будет закрыт даже для пользователей с правами root и администраторов. Эту настройку нельзя переопределить для отдельного контейнера.
:::

## Права доступа к терминалу
Начиная с версии **v4.0.0-beta.452**, только пользователи с правами **root** и **администраторы** (admin) могут изменять настройки доступа к терминалу.

Когда доступ к терминалу отключен:
- Никто не может получить доступ к терминалу сервера.
- Никто не может получить доступ к терминалу любого контейнера на этом сервере.
- Ограничение распространяется даже на root-пользователей и администраторов.
- Изменения вступают в силу немедленно.

Вот и всё! Теперь вы полностью контролируете доступ к терминалу для ваших серверов и контейнеров.
