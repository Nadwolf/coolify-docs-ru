# Обновление сервера (Server Patching)
Функция **Обновление сервера** позволяет обновлять все пакеты на вашем сервере прямо из панели управления Coolify.

::: warning ВАЖНО
Эта функция появилась в версии **v4.0.0-beta.419**. Чтобы следовать этому руководству, вы **должны** использовать версию v4.0.0-beta.419 или выше.
:::

## Как использовать обновление сервера
<ZoomableImage src="/docs/images/knowledge-base/servers/patching/1.webp" alt="скриншот обновления" />
1. Перейдите в раздел **Servers** в боковой панели.
2. Выберите ваш сервер.
3. Нажмите на вкладку **Security**.
4. Откройте раздел **Server Patching**.

На этой странице вы увидите список всех доступных обновлений пакетов.

Вы можете:
  - Обновить пакеты **по отдельности**, нажав кнопку **Update** рядом с каждым из них.
  - Или нажать **Update All Packages**, чтобы применить все обновления сразу.

::: info Примечание
  Coolify **не** устанавливает обновления автоматически. Система только проверяет наличие доступных обновлений и отображает их в дашборде.
  
  Обновления применяются только после того, как вы вручную нажмете кнопку **Update** или **Update All Packages**.
:::

::: danger Важно
Обновление некоторых пакетов может привести к нарушению работы существующих функций.
Обновления, связанные с Docker, перезапустят Docker, что означает временную недоступность всех ваших приложений и самого Coolify до завершения перезапуска.

**Мы настоятельно рекомендуем внимательно проверять каждый пакет перед обновлением.**
:::

## Уведомления об обновлениях
Coolify проверяет наличие обновлений еженедельно и отправляет уведомления, если они доступны.

  - Уведомления **включены по умолчанию**.
  - Вы можете изменить это в разделе **Notification Settings** (Настройки уведомлений).

<ZoomableImage src="/docs/images/knowledge-base/servers/patching/2.webp" alt="скриншот обновления" />

## Поддерживаемые менеджеры пакетов
Начиная с версии **v4.0.0-beta.419**, функция обновления сервера поддерживается для следующих менеджеров пакетов:

* **APT**
* **DNF**
* **Zypper**

Поддержка других менеджеров пакетов будет добавлена в будущих обновлениях.

Если вы хотите приоритетную поддержку конкретного менеджера пакетов, вы можете начать обсуждение здесь: [Coolify Discussions на GitHub](https://github.com/coollabsio/coolify/discussions)

## FAQ (Часто задаваемые вопросы)

::: details 1. Можно ли изменить частоту проверки обновлений Coolify?
Нет, на данный момент интервал проверки фиксирован.

Однако вы можете запустить проверку вручную, нажав кнопку **Check now** на странице Server Patching.

Мы рассматриваем возможность настройки интервалов в будущем на основе отзывов пользователей. Если вам нужна такая функция, вы можете написать об этом в [обсуждениях Coolify на GitHub](https://github.com/coollabsio/coolify/discussions).
:::

::: details 2. Устанавливает ли Coolify обновления автоматически?
**Нет**, Coolify не устанавливает пакеты автоматически.

Обновления применяются только после вашего ручного нажатия кнопки **Update** или **Update all packages** в дашборде.
:::
