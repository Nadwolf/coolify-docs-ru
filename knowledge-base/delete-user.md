# Удаление пользователя
Это руководство покажет вам, как удалить пользователя из вашего self-hosted инстанса Coolify.

::: warning Осторожно
  Если вы удаляете пользователя и у Coolify всё еще есть доступ к серверам, он также удалит все ресурсы на этих серверах, а не только записи из базы данных Coolify.
:::

Только пользователи, находящиеся в команде `root`, могут удалять других пользователей через интерфейс.

- Перейдите на страницу `Team`.
- Переключитесь на вкладку `Admin View`.

Здесь вы увидите всех зарегистрированных в Coolify пользователей. Нажмите кнопку `Delete` рядом с пользователем, которого хотите удалить.

## Процесс удаления

::: info Совет
  Нельзя удалить команду root или пользователя root.
:::

Coolify проходит по всем командам пользователя и принимает следующие решения:

### Пользователь единственный в команде

Команда и все ресурсы на сервере, а также в базе данных Coolify, будут удалены.

### В команде есть другие пользователи

1. **Пользователь является владельцем/админом команды**, других владельцев нет, но в команде **есть участники**:
   - Права владения передаются первому участнику команды, который не является владельцем/админом.
   - Пользователь удаляется из команды. Ресурсы не удаляются.

2. **Пользователь является владельцем/админом команды**, других владельцев нет, и в команде **нет участников**:
   - Команда и все ресурсы на сервере, а также в базе данных Coolify, будут удалены.

3. **Пользователь не является владельцем/админом команды**:
   - Пользователь удаляется из команды. Ресурсы не удаляются.
