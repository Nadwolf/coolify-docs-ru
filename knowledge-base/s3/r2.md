# Cloudflare R2
Cloudflare R2 — это S3-совместимое хранилище. Вы можете использовать его в Coolify для хранения ваших резервных копий.

# Конфигурация

1. Сначала вам нужно создать бакет в панели управления Cloudflare R2.
2. Затем создайте токен API R2 с правами `Object Read & Write` (Чтение и запись объектов).
3. При создании токена вы получите учетные данные для S3-клиента.
   ::: success Совет
     Вам понадобятся `Access Key ID`, `Secret Access Key` и `S3 endpoint` из этого окна. Сохраните их.
   :::
4. Используйте полученные данные для настройки S3-хранилища в Coolify.
