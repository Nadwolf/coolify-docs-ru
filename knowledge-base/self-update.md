# Обновление инстанса Coolify (Self-hosted)

## Настройки обновления

Вы можете настроить параметры обновления вашего локального/self-hosted инстанса Coolify на странице `Settings` (Настройки) в разделе `Update`.

Существует две основные конфигурации обновления:

1. **Частота проверки обновлений (Update Check Frequency)**
   - Определяет, как часто Coolify проверяет наличие:
     - Новых версий самого приложения Coolify.
     - Новых шаблонов сервисов из официальной CDN ([one-click services]).
   - По умолчанию: проверяет каждый час.
   - Используется синтаксис cron для гибкой настройки времени.

2. **Частота автоматического обновления (Auto Update Frequency)**
   - Определяет, в какое время Coolify будет автоматически устанавливать новые обновления (если они найдены).
   - По умолчанию: загружает и ставит ежедневно в полночь (00:00).
   - Используется синтаксис cron для гибкой настройки времени.

## Переключатель автообновления

- По умолчанию для self-hosted инстансов опция автоматического обновления (`Auto Update Enabled`) **включена**.
- Вы можете без проблем отключить автоматические обновления (просто выключив чекбокс), что строго рекомендуется для критически важных продакшн-окружений. Обратите внимание, что отключение автоматического обновления никак не отменяет проверку на наличие новых версий — она все равно продолжит выполняться в фоне периодически.
- Если вы отключите автообновление, вы все равно всегда сможете обновить Coolify вручную в один клик мыши, нажав на кнопку `Update`, которая появится, когда станет доступна новая версия.

## Настройка расписаний обновлений

Оба параметра (и частота проверки, и частота установки) используют мощный синтаксис cron для планирования задач. Подробную информацию о поддерживаемом синтаксисе см. в нашем [руководстве по синтаксису cron (Cron Syntax)](/knowledge-base/cron-syntax).

## Доступность новых версий

Чтобы узнать, что нового появилось в свежих релизах, исправления багов, подробную информацию о доступности и схеме версионирования, пожалуйста, читайте файл [RELEASE.md](https://github.com/coollabsio/coolify/blob/main/RELEASE.md) в нашем главном репозитории на GitHub.
