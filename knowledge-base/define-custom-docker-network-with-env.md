# Настройка пользовательской сети Docker через переменные окружения

## Требования к валидации

Для пользовательской сети Docker в продакшн-окружении должны быть соблюдены следующие требования.

### База пула адресов (`DOCKER_ADDRESS_POOL_BASE`)
- Должна быть валидным блоком CIDR, например `10.0.0.0/8`.

### Размер пула адресов (`DOCKER_ADDRESS_POOL_SIZE`)
- Должен быть валидным числом, например `10`.

### Принудительная перезапись (`DOCKER_POOL_FORCE_OVERRIDE`)
- Требуется только в том случае, если на хосте уже есть пул адресов Docker и вы хотите его перезаписать.

## Автоматизированный метод установки

1. **Подготовьте параметры**
   Создайте параметры сети в соответствии с требованиями выше.

2. **Запустите команду установки**
   Выполните автоматизированный скрипт установки с вашими параметрами:

   ```bash
   env DOCKER_ADDRESS_POOL_BASE=10.0.0.0/8 DOCKER_ADDRESS_POOL_SIZE=10 bash -c 'curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash'
   ```
   > Посмотреть [исходный код скрипта](https://github.com/coollabsio/coolify/blob/main/scripts/install.sh)

::: info
  Скрипт установки должен запускаться от имени `root`. Если вы не под `root`, используйте `sudo` для повышения привилегий.
  ```bash
  sudo -E env DOCKER_ADDRESS_POOL_BASE=10.0.0.0/8 DOCKER_ADDRESS_POOL_SIZE=10 bash -c 'curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash'
  ```
:::

3. **Завершите настройку**
   После настройки параметров сети продолжите выполнение [шагов установки](/get-started/installation#quick-installation-recommended) для завершения настройки Coolify.

## Ручной метод установки

1. **Подготовьте параметры**
   Создайте параметры сети в соответствии с требованиями выше.

2. **Настройте переменные окружения**
   Отредактируйте файл переменных окружения:

   ```bash
   nano /data/coolify/source/.env
   ```

   Добавьте следующие переменные:
   ```bash
   DOCKER_ADDRESS_POOL_BASE=10.0.0.0/8
   DOCKER_ADDRESS_POOL_SIZE=10
   DOCKER_POOL_FORCE_OVERRIDE=false
   ```

3. **Завершите настройку**
   После настройки параметров сети продолжите выполнение [шагов установки](/get-started/installation#manual-installation) для завершения настройки Coolify.
