# OAuth

Вы можете войти в панель управления Coolify традиционным способом, используя email и пароль, или с помощью технологии OAuth.

Используя OAuth, вы передаете функцию аутентификации внешнему провайдеру (Identity Provider, IDP). Это позволяет серверу Coolify гарантированно подтвердить, что новый (или уже зарегистрированный в инстансе) пользователь действительно владеет конкретным адресом электронной почты, связанным с его профилем. По сути, это безопасная альтернатива ручному вводу пароля на странице регистрации или входа в Coolify.

**В настоящее время поддерживаются следующие провайдеры:**
- Azure
- Bitbucket
- GitHub
- GitLab
- Google

## Настройка OAuth

Для успешной настройки OAuth вам необходимо предварительно создать соответствующее приложение у выбранного провайдера авторизации. После создания вы получите `Client ID` и `Client Secret` (или аналогичные им секретные ключи). Их необходимо будет добавить в настройках вашего инстанса.

Перейдите по адресу: `ВАШ_ПРОЕКТ_COOLIFY/settings/oauth`.

Кроме того, вам обязательно необходимо указать параметр `Redirect URI` в панели управления провайдера авторизации. Это позволит стороннему сервису отправить авторизованные данные пользователя (токен доступа) обратно в Coolify, чтобы подтвердить, что доступ предоставлен корректно.

`Redirect URI` всегда должен быть в следующем формате:
`ВАШ_ПРОЕКТ_COOLIFY/auth/НАЗВАНИЕ_ПРОВАЙДЕРА/callback`.

Например: `https://coolify.example.com/auth/google/callback`.

### Примеры настройки популярных провайдеров:

- [Google OAuth](https://support.google.com/cloud/answer/6158849?hl=ru)
  - В поле `Authorized JavaScript origins` (Разрешенные источники JavaScript) введите: `ВАШ_ПРОЕКТ_COOLIFY`.
  - В поле `Authorized redirect URIs` (Разрешенные URI перенаправления) введите URI, который вы указали в настройках OAuth в Coolify для Google. Например: `https://coolify.example.com/auth/google/callback`.

- [GitHub OAuth](https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/creating-an-oauth-app)
  - В поле `Homepage URL` (Адрес главной страницы) введите: `ВАШ_ПРОЕКТ_COOLIFY`.
  - В поле `Authorization callback URL` (URL перенаправления авторизации) введите URI, который вы указали в настройках OAuth в Coolify для GitHub. Например: `https://coolify.example.com/auth/github/callback`.
