# Caddy Basic Auth

Базовая аутентификация (Basic Auth) обеспечивает дополнительный уровень безопасности для ваших приложений, требуя ввода имени пользователя и пароля для доступа к защищенным ресурсам.

В Coolify вы можете легко интегрировать базовую аутентификацию в конфигурацию веб-сервера Caddy.

## Зачем использовать Basic Auth в Caddy?

1. **Повышенная безопасность:** добавляет дополнительный барьер для предотвращения несанкционированного доступа.
2. **Простота:** прямолинейная конфигурация, которая интегрируется непосредственно в настройки Caddy.
3. **Гибкость:** возможность настройки различных учетных данных для разных сервисов по мере необходимости.

## 1. Генерация хешированного пароля

Для безопасной проверки учетных данных Caddy требует, чтобы ваш пароль был захеширован с помощью встроенного инструмента Caddy. Запись учетных данных выглядит так:

```sh
caddy_0.basicauth.<username>="<hashed_password>"
```

`<hashed_password>` **должен быть сгенерирован через Caddy CLI** с помощью команды `caddy hash-password`.

### Как сгенерировать хешированный пароль

1. Откройте терминал.
2. Выполните следующую команду:

   ```sh
   caddy hash-password --plaintext "ваш_пароль"
   ```

   Замените `"ваш_пароль"` на ваш реальный пароль.

3. Выводом команды будет хешированный пароль, который можно использовать в конфигурации Caddy.

Для получения более подробной информации и расширенных опций (например, выбора другого алгоритма) обратитесь к [документации Caddy CLI](https://caddyserver.com/docs/command-line#caddy-hash-password?utm_source=coolify.io).

## 2. Настройка Basic Auth в Coolify

Когда у вас есть хешированный пароль, добавьте его в конфигурацию Coolify.

1. **Добавьте запись Basic Auth:**

   - Добавьте следующую строку в Caddyfile приложения, для которого вы хотите включить аутентификацию:
     ```sh
     caddy_0.basicauth.<username>="<hashed_password>"
     ```
   - Замените `<username>` на желаемое имя пользователя, а `<hashed_password>` на результат выполнения команды `caddy hash-password`.

2. **Примените конфигурацию:**
   - Сохраните изменения.
   - Перезапустите приложение, чтобы настройки вступили в силу.

::: warning Примечание
Убедитесь, что хешированный пароль сгенерирован **именно** с помощью Caddy CLI.

Использование неподдерживаемых методов хеширования приведет к ошибкам аутентификации.
:::
