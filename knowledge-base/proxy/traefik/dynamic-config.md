# Динамические конфигурации
Динамические конфигурации позволяют настраивать Traefik на лету, не перезапуская его.

Вы можете добавить их, перейдя в раздел `Server` -> `Proxy` и выбрав `Dynamic Configurations` в боковом меню.

::: tip Совет
  Некоторые динамические конфигурации нельзя удалить, так как они необходимы для работы самого Coolify.
:::
