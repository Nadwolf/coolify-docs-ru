# Использование WordPress Multisite в Coolify

1. **Добавьте WordPress через установку в один клик**
   Добавьте сервис WordPress, используя функцию установки "в один клик" в Coolify.

2. **Настройте хранение файлов на хост-машине**
   Измените следующие строки в вашем файле `docker-compose.yml` через интерфейс Coolify, чтобы файлы WordPress хранились на сервере:
    ```yaml
    volumes:
      - "wordpress-files:/var/www/html"
    ```
    на:
    ```yaml
    volumes:
      - "./wordpress:/var/www/html"
    ```
    Это примонтирует директорию `wordpress` из папки конфигурации по умолчанию (`/data/coolify/services/<serviceUuid>/`) в директорию `/var/www/html` внутри контейнера. Таким образом, вы сможете редактировать файлы на хост-машине, и изменения сразу вступят в силу в контейнере.

3. **Настройте WordPress**
    Запустите сервис WordPress и выполните базовую настройку.

4. **Отключите все плагины**
    - Зайдите в панель администратора WordPress.
    - Перейдите в раздел `Plugins` -> `Installed Plugins`.
    - Выберите все плагины и выберите `Deactivate` (Деактивировать) в выпадающем меню.

5. **Включите режим Multisite**
    Откройте файл `wp-config.php` на сервере и добавьте следующую строку:
    ```php
      define( 'WP_ALLOW_MULTISITE', true );
    ```
    Обновите страницу панели WordPress в браузере. В меню `Tools` (Инструменты) должен появиться новый пункт `Network Setup` (Настройка сети).
