# Приватный реестр NPM
Если вы хотите использовать приватный реестр NPM с Coolify, выполните следующие шаги:

1. Добавьте файл `.npmrc` в корень вашего проекта со следующим содержимым:

```bash
//registry.npmjs.org/:_authToken=${NPM_TOKEN}
```

2. Добавьте следующую переменную окружения в ваш проект как переменную **сборки** (build variable):

```bash
NPM_TOKEN=ваш_npm_токен
```

3. Разверните ваше приложение.
