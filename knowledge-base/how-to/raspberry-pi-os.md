# Руководство по настройке Raspberry Pi OS
## Предварительные требования

Для запуска Coolify на Raspberry Pi вам понадобится одна из следующих моделей:
  - Raspberry Pi Zero 2 W
  - Raspberry Pi 400
  - Raspberry Pi 3 (все модели)
  - Raspberry Pi 4 (все модели)
  - Raspberry Pi 5 (все модели)

## Установка

1. Скачайте и установите программу [Raspberry Pi Imager](https://www.raspberrypi.com/software/) на ваш компьютер.

2. Вставьте карту microSD в картридер вашего компьютера.

3. Откройте Raspberry Pi Imager и выберите ваше устройство:
   - Нажмите `Choose Device`.
   - Выберите вашу модель Raspberry Pi.

4. Выберите операционную систему:
   - Нажмите `Choose OS`.
   - Перейдите в раздел `Raspberry Pi OS (other)`.
   - Выберите `Raspberry Pi OS Lite (64-bit)`.

   ::: warning Осторожно
   Вы ДОЛЖНЫ выбрать одну из 64-битных версий ОС, так как Coolify не совместим с 32-битными версиями.
   :::

   ::: info Примечание
   Хотя вы можете использовать полную версию с рабочим столом `Raspberry Pi OS (64-bit)` / `Raspberry Pi OS Full (64-bit)` или даже `Ubuntu`, мы рекомендуем версию `Raspberry Pi OS Lite`, так как она потребляет меньше ресурсов.
   :::

5. Выберите накопитель:
   - Нажмите `Choose Storage`.
   - Выберите вашу карту microSD.
   - Дважды проверьте, что выбрали правильный диск, чтобы избежать потери данных.

6. Нажмите `Next` и выберите `Edit settings` для настройки ОС.
   - Перейдите в раздел `Services` и включите SSH с использованием публичного ключа.

   ::: warning Осторожно
   SSH-ключ не должен иметь кодовой фразы (passphrase) или включенной двухфакторной аутентификации (2FA), иначе вы не сможете завершить процесс настройки Coolify.
   :::

   - Настройте другие параметры по мере необходимости.

7. Завершите процесс записи на SD-карту.

8. После завершения вставьте microSD в Raspberry Pi и включите питание.

9. Когда Raspberry Pi загрузится, приступайте к [установке Coolify](/get-started/installation#quick-installation-recommended).
