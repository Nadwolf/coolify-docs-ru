# Ollama с поддержкой GPU
На основе подробного руководства от [geek.sg](https://geek.sg/blog/how-i-self-hosted-llama-32-with-coolify-on-my-home-server-a-step-by-step-guide?ref=coolify.io):

1. **Аппаратные требования**
   - Сервер с NVIDIA GPU (тестировалось на RTX 3060 12GB).
   - Рекомендуется минимум 32 ГБ оперативной памяти.
   - Достаточное количество свободного места для моделей.

2. **Программная настройка**
   - Установка драйверов NVIDIA.
   - Установка NVIDIA Container Toolkit.
   - Настройка Docker для использования рантайма NVIDIA.

3. **Конфигурация в Coolify**
   - Развертывание Ollama через установщик расширений "в один клик" в Coolify.
   - Изменение конфигурации Docker Compose для включения поддержки GPU.
   - Добавление необходимых переменных окружения для ускорения GPU.

4. **Управление моделями**
   - Загрузка (pull) и управление вашими любимыми моделями LLM.
   - Мониторинг использования и производительности GPU.
   - Настройка параметров моделей при необходимости.

Для получения полной версии руководства посетите [оригинальную статью](https://geek.sg/blog/how-i-self-hosted-llama-32-with-coolify-on-my-home-server-a-step-by-step-guide?ref=coolify.io).
