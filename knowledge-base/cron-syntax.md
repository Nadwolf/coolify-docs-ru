# Поддерживаемый синтаксис Cron
Coolify поддерживает планирование автоматических задач, таких как очистка, резервное копирование и многое другое, с помощью синтаксиса cron.

## Поддерживаемый синтаксис

### Стандартный формат Cron
Coolify поддерживает полный стандартный формат синтаксиса cron (`* * * * *`).

### Предустановленные расписания
Для удобства Coolify также поддерживает следующие предустановленные строки расписания:

#### Без префикса @
- `every_minute` — запускается каждую минуту.
- `hourly` — запускается раз в час.
- `daily` — запускается раз в день.
- `weekly` — запускается раз в неделю.
- `monthly` — запускается раз в месяц.
- `yearly` — запускается раз в год.

#### С префиксом @
- `@every_minute` — запускается каждую минуту.
- `@hourly` — запускается раз в час.
- `@daily` — запускается раз в день.
- `@weekly` — запускается раз в неделю.
- `@monthly` — запускается раз в месяц.
- `@yearly` — запускается раз в год.
