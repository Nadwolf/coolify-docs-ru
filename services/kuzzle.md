## Что такое Kuzzle?

Kuzzle — это универсальный бэкенд, предлагающий базовые строительные блоки (building blocks), общие для каждого приложения.

Вместо того чтобы раз за разом разрабатывать одни и те же стандартные функции при создании каждого нового приложения, Kuzzle предлагает их в готовом виде (off the shelf), позволяя вам сосредоточиться на создании высокоуровневых и ценных бизнес-функций.

## Возможности (Features)

Kuzzle позволяет вам создавать современные веб-приложения и сложные сети IoT в кратчайшие сроки.

- **API First**: используйте стандартизированный многопротокольный (multi-protocol) API.
- **Постоянные данные (Persisted Data)**: сохраняйте ваши данные и выполняйте по ним расширенный поиск.
- **Уведомления в реальном времени (Realtime Notifications)**: используйте систему pub/sub или подписывайтесь (subscribe) на уведомления базы данных.
- **Управление пользователями (User Management)**: вход (login), выход (logout) и правила безопасности больше не являются обузой.
- **Расширяемость (Extensible)**: разрабатывайте расширенные бизнес-функции напрямую с помощью интегрированного фреймворка.
- **Клиентские SDK (Client SDKs)**: используйте наши SDK для ускорения разработки фронтенда.
