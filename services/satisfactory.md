# Что такое Satisfactory?

Satisfactory — это игра от первого лица в открытом мире, представляющая собой симулятор фабрики, сосредоточенный на строительстве, автоматизации и исследовании чужой планеты. Игроки, выступая в роли «Пионеров капитала» компании FICSIT Inc., исследуют мир, собирают ресурсы и строят массивные многоэтажные заводы, соединенные конвейерными лентами для автоматизации производства.

## Информация (Info)

Сервер может работать менее чем на 8 ГБ оперативной памяти, хотя рекомендуется от 8 ГБ до 16 ГБ согласно [официальной вики](https://satisfactory.wiki.gg/wiki/Dedicated_servers#Requirements?utm_source=coolify.io). Вам может потребоваться увеличить ограничение `--memory` контейнера по мере приближения к поздним этапам игры (или если вы играете большой группой более 4 игроков).

### Обновление (Updating)

Игра автоматически обновляется при запуске или перезапуске контейнера (если вы не установите `SKIPUPDATE=true`).

### Переменные окружения (Environment Variables)

| Параметр               |  По умолчанию  | Функция                                                       |
|-------------------------|:---------:|:--------------------------------------------------------------|
| `AUTOSAVENUM`           |    `5`    | количество файлов циклического автосохранения                 |
| `DEBUG`                 |  `false`  | для отладки сервера                                           |
| `DISABLESEASONALEVENTS` |  `false`  | отключить сезонные события FICSMAS                            |
| `LOG`                   |  `false`  | отключить очистку (pruning) логов Satisfactory                |
| `MAXOBJECTS`            | `2162688` | установить лимит объектов для вашего сервера                  |
| `MAXPLAYERS`            |    `4`    | установить лимит игроков для вашего сервера                   |
| `MAXTICKRATE`           |   `30`    | установить максимальную частоту тиков симуляции (tick rate)    |
| `MULTIHOME`             |   `::`    | установить интерфейс прослушивания сервера (обычно не нужно)  |
| `PGID`                  |  `1000`   | установить ID группы пользователя, от которого работает сервер|
| `PUID`                  |  `1000`   | установить ID пользователя, от которого работает сервер       |
| `SERVERGAMEPORT`        |  `7777`   | установить игровой порт сервера                               |
| `SERVERMESSAGINGPORT`   |  `8888`   | установить порт обмена сообщениями (внутренний и внешний)     |
| `SERVERSTREAMING`       |  `true`   | переключить использование потоковой передачи ресурсов (streaming)|
| `SKIPUPDATE`            |  `false`  | не обновлять игру при старте/перезапуске контейнера           |
| `STEAMBETA`             |  `false`  | установить экспериментальную версию игры                      |
| `STEAMBETAID`           |     -     | установить кастомную бета-версию игры (для тестирования)      |
| `STEAMBETAKEY`          |     -     | установить пароль для бета-версии игры (для тестирования)     |
| `TIMEOUT`               |   `30`    | установить таймаут клиента (в секундах)                       |
| `VMOVERRIDE`            |  `false`  | пропустить проверку модели процессора (обычно не используется)|
