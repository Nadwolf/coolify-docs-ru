## Что такое Convex

Convex — это open-source реактивная база данных для разработчиков приложений.

## Как сгенерировать токен администратора (Admin Key)

Чтобы сгенерировать Admin Key для вашего приложения Convex в Coolify, выполните следующие шаги:

1. Перейдите в панель управления Coolify.
2. Выберите ваше приложение Convex.
3. Откройте вкладку **Terminal**.
4. Подключитесь к терминалу бэкенда.
5. В терминале выполните следующую команду: `./generate_admin_key.sh`
6. Скопируйте сгенерированный администраторский ключ.
