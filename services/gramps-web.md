# Gramps Web

![Логотип Gramps Web](https://coolify.io/docs/images/services/grampsweb-logo.svg){ width=64px }

## Что такое Gramps Web?

Бесплатная open-source генеалогическая система для совместного построения семейного древа. Используйте ее как самостоятельное приложение (standalone) или как дополнение к Gramps Desktop с полным контролем над вашими данными и с упором на конфиденциальность (privacy).
