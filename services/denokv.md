## Что такое Deno KV

Deno KV — это «JavaScript-first» база данных типа «ключ-значение» с возможностью self-hosting, встроенная непосредственно в среду выполнения Deno. Она предоставляет надежное и высокопроизводительное решение для хранения данных на базе FoundationDB и SQLite, разработанное для современных edge-вычислений и серверлесс-приложений.

Ключевые особенности включают:

- **ACID-транзакции**: Гарантированная согласованность данных с полной поддержкой ACID-транзакций, предотвращающая частичную запись и повреждение данных.
- **Построено на FoundationDB**: Бэкенд корпоративного уровня, способный обрабатывать миллионы операций в секунду.
- **Watch API в реальном времени**: Мониторинг изменений в базе данных в реальном времени для создания реактивных приложений, уведомлений и живых обновлений.
- **Нулевая конфигурация**: Работает «из коробки» (out of the box) вместе с Deno runtime, не требуя сложной настройки.
- **Нативный JavaScript**: Храните любые JavaScript-значения напрямую без ручной сериализации.
- **Иерархические ключи**: Организуйте данные с помощью структурированных, похожих на REST шаблонов ключей для интуитивного моделирования данных.
- **Несколько уровней согласованности**: Балансируйте между производительностью и согласованностью в зависимости от потребностей вашего приложения.
- **Self-hosted и Open Source**: Лицензия MIT с полным контролем над вашей инфраструктурой.
