## Что такое NextCloud?

NextCloud — это open-source платформа для повышения продуктивности, которая предоставляет безопасную и приватную альтернативу популярным сервисам облачного хранения (cloud storage), таким как Google Drive и Dropbox. Она позволяет вам хранить, синхронизировать и обмениваться файлами, фотографиями и видео на нескольких устройствах, и предлагает такие функции, как управление версиями файлов (file versioning), защищенный паролем обмен файлами и инструменты для совместной работы.

## Варианты развертывания

Nextcloud доступен в четырех конфигурациях развертывания в Coolify:

### Nextcloud (По умолчанию)
- **База данных:** SQLite (встроенная)
- **Сценарий использования:** Простые развертывания, тестирование или личное хранилище файлов
- **Компоненты:** Единый контейнер Nextcloud со встроенной базы данных SQLite

### Nextcloud с PostgreSQL
- **База данных:** PostgreSQL
- **Сценарий использования:** Prod-развертывания (production), требующие лучшей производительности и масштабируемости
- **Компоненты:**
  - Контейнер Nextcloud
  - Контейнер PostgreSQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Nextcloud с MySQL
- **База данных:** MySQL
- **Сценарий использования:** Prod-развертывания с предпочтением использования MySQL
- **Компоненты:**
  - Контейнер Nextcloud
  - Контейнер MySQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Nextcloud с MariaDB
- **База данных:** MariaDB
- **Сценарий использования:** Prod-развертывания с предпочтением использования MariaDB (рекомендуется для большинства пользователей)
- **Компоненты:**
  - Контейнер Nextcloud
  - Контейнер MariaDB
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)
