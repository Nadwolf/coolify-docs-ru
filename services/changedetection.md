## Что такое Change Detection?

Обнаруживайте изменения контента веб-сайтов и выполняйте полезные действия — запускайте уведомления через Discord, Email, Slack, Telegram, вызовы API и многое другое.

Живите проактивной жизнью с данными.

## Примеры использования

- Изменение цен на продукты и услуги
- Уведомления *Товар распродан (Out of stock)* и *Снова в наличии (Back In stock)*
- Мониторинг и отслеживание изменений в PDF-файлах: узнавайте, когда в PDF-файле изменился текст.
- Обновления от государственных ведомств (часто изменения публикуются только на их сайтах)
- Новые релизы программного обеспечения, рекомендации по безопасности, когда вас нет в их списке рассылки.
- Изменения на фестивалях
- Уведомления о поступлении товаров на Discogs и мониторинг
- Изменения в объявлениях о недвижимости
- Узнавайте, когда на ваш любимый виски скидка или о других специальных предложениях раньше всех
- Новости, связанные с COVID, с правительственных сайтов
- Новости университетов/организаций с их сайтов
- Обнаружение и мониторинг изменений в ответах JSON API
- Мониторинг JSON API и алертинг (оповещения)
- Изменения в юридических и других документах
- Запуск вызовов API через уведомления (notifications), когда на сайте появляется текст
- Склейка (Glue together) API с использованием фильтра JSON и уведомлений JSON
- Создание RSS-каналов на основе изменений веб-контента
- Мониторинг исходного кода HTML на наличие неожиданных изменений, усиление вашей защиты и совместимости с PCI
- У вас есть очень конфиденциальный список URL-адресов для отслеживания, и вы *не* хотите использовать платные альтернативы. (Помните: *вы* — это товар)
- Получайте уведомления, когда в результатах поиска Twitter появляются определенные ключевые слова
- Проактивно ищите работу: получайте уведомления, когда компании обновляют страницу с вакансиями (careers page), или при появлении нужных слов (keywords) на порталах вакансий.
- Получайте уведомления, когда открываются новые вакансии на Bamboo HR и других платформах поиска работы
- Мониторинг дефейсмента (взлома и подмены содержимого) сайтов
- Отслеживание поступлений карточек Pokémon (Pokémon TCG Tracker)
- RegTech — будьте в курсе нормативных изменений (regulatory changes) и соблюдения требований (regulatory compliance)

## Ключевые особенности

- Множество фильтров (trigger filters), таких как "Запуск по тексту (Trigger on text)", "Удаление текста по селектору (Remove text by selector)", "Игнорирование текста", "Извлечение текста", а также с использованием регулярных выражений!
- Анализ нужных элементов с помощью xPath (1.0) и CSS-селекторов, легкий мониторинг сложных JSON с помощью JSONPath или jq
- Переключение между быстрыми «сборщиками (fetchers)» без JS и сборщиками на основе Chrome JS
- Отслеживание изменений в PDF-файлах (Отслеживайте измененный текст в PDF, а также контролируйте размер PDF-файла и контрольные суммы)
- Легко указывайте, как часто следует проверять сайт
- Выполнение (Execute) JS перед извлечением текста (Отлично подходит для авторизации (logging in), смотрите примеры в интерфейсе!)
- Переопределение заголовков запросов (Request Headers), указание `POST`, `GET` или других методов (methods)
- Используйте «Визуальный селектор» (Visual Selector), чтобы помочь нацелиться на нужные элементы
- Настраиваемый [прокси для каждого отслеживания (proxy per watch)](https://github.com/dgtlmoon/changedetection.io/wiki/Proxy-configuration?utm_source=coolify.io)
- Отправка скриншота с уведомлением (notification) при обнаружении изменения на странице.
