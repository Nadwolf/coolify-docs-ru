## Что такое Gitea?

Git с чашечкой чая! Безболезненный self-hosted all-in-one сервис для разработки программного обеспечения, включающий Git-хостинг, ревью кода (code review), совместную работу команды, реестр пакетов (package registry) и CI/CD.

## Варианты развертывания

Gitea доступна в четырех конфигурациях развертывания в Coolify:

### Gitea (По умолчанию)
- **База данных:** SQLite (встроенная)
- **Сценарий использования:** Простые развертывания, личные проекты или тестирование
- **Компоненты:** Единый контейнер Gitea со встроенной базой данных SQLite

### Gitea с PostgreSQL
- **База данных:** PostgreSQL
- **Сценарий использования:** Prod-развертывания (production), требующие совместимости с PostgreSQL и лучшей производительности
- **Компоненты:**
  - Контейнер Gitea
  - Контейнер PostgreSQL 16
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Gitea с MySQL
- **База данных:** MySQL
- **Сценарий использования:** Prod-развертывания с предпочтением MySQL
- **Компоненты:**
  - Контейнер Gitea
  - Контейнер MySQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Gitea с MariaDB
- **База данных:** MariaDB
- **Сценарий использования:** Prod-развертывания с предпочтением MariaDB
- **Компоненты:**
  - Контейнер Gitea
  - Контейнер MariaDB
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)
