## Что такое Terraria?

Terraria — это 2D-приключенческая игра-песочница, в которой вы можете копать, сражаться, исследовать и строить в огромном процедурно генерируемом мире. Игроки могут сражаться с врагами, находить сокровища, создавать оружие и броню, а также строить сооружения в различных биомах.

## Переменные окружения (Environment Variables)

| Имя        | Описание                                                                                                                                                                                                         | Обязательно | Значение по умолчанию   |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ---------------------- |
| AUTOCREATE | Создает мир, если он не найден по пути, указанному в `-world`. Размер мира: 1 (маленький), 2 (средний) и 3 (большой).                                                                                    | да      | 2                      |
| WORLDNAME  | Устанавливает имя мира при использовании `-autocreate`.                                                                                                                                                                  | да      | world1                 |
| DIFFICULTY | Устанавливает сложность мира при использовании `-autocreate`. Варианты: 0 (обычный), 1 (эксперт), 2 (мастер), 3 (путешествие)                                                                                    | да      | 1                      |
| MAXPLAYERS | Максимально допустимое количество игроков                                                                                                                                                                               | да      | 8                      |
| PASSWORD   | Установить пароль для сервера                                                                                                                                                                                       | да      | mypassword             |
| MOTD       | Установить текст приветствия сервера (сообщение дня).                                                                                                                                                                               | да      | Welcome to the server! |
| LANGUAGE   | Устанавливает язык сервера по его коду. Доступные коды: en/US = английский, de/DE = немецкий, it/IT = итальянский, fr/FR = французский, es/ES = испанский, ru/RU = русский, zh/Hans = китайский, pt/BR = португальский, pl/PL = польский | да      | en/US                  |
| SECURE     | Опция для предотвращения читов. (1: без читов или 0: читы разрешены)                                                                                                                                                       | да      | 1                      |
