## Что такое GitHub Runner?

GitHub Runner — это self-hosted раннер (runner) для GitHub Actions, который позволяет запускать рабочие процессы CI/CD в вашей собственной инфраструктуре. Это решение на базе Docker обеспечивает больший контроль над средой выполнения (execution environment), лучшую безопасность для приватных репозиториев и может быть более экономичным для тяжелых рабочих нагрузок (heavy workloads). Оно поддерживает запуск рабочих процессов для нескольких репозиториев и организаций.
