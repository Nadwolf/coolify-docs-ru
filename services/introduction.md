# Сервисы

## Что такое Сервисы?

Сервисы (Services) в Coolify — это развертывания, основанные на **файлах Docker Compose**, которые хранятся непосредственно на вашем сервере. В отличие от [Приложений (Applications)](/applications/introduction), Сервисы **не связаны с источником Git** — они не скачивают код из репозитория и не пересобираются при коммитах.

В Coolify есть два типа Сервисов:

### Пользовательские Сервисы (User-Defined Services)

Это файлы Docker Compose, которые **вы предоставляете вручную**. Вы можете вставить свою конфигурацию `docker-compose.yml` прямо в Coolify, и он развернет ее и будет ею управлять. Это дает вам полную гибкость для развертывания любого контейнерного приложения или стека, который вы настроили сами.

Для этого при создании нового ресурса выберите опцию `Docker Compose Empty` (Пустой Docker Compose).

### Сервисы в один клик (One-Click Services)

Сервисы в один клик — это отобранная (curated) коллекция из более чем 200 популярных open-source приложений и инструментов, которые вы можете развернуть мгновенно всего за несколько кликов. Это **предварительно настроенные шаблоны Docker Compose**, предоставляемые Coolify, которые устраняют сложность ручной установки и настройки.

Вместо того чтобы писать собственный файл compose с нуля, вы можете выбрать один из готовых к использованию шаблонов, которые автоматически заполнятся за вас, делая self-hosting доступным для всех.

#### Как они работают

Каждый шаблон был:

- **Предварительно протестирован и оптимизирован** для надежного развертывания
- **Настроен с разумными значениями по умолчанию**, чтобы работать прямо "из коробки" (out of the box)
- **Интегрирован с функциями Coolify**, такими как автоматические SSL-сертификаты и бэкапы

#### Что включено

Наша библиотека сервисов включает в себя широкий спектр приложений, разработанных open-source сообществом:

- **Инструменты разработки (Development Tools)**: Редакторы кода, Git-платформы, решения для CI/CD
- **Приложения для продуктивности (Productivity Apps)**: Управление проектами, создание заметок, инструменты для совместной работы
- **Медиа и развлечения (Media & Entertainment)**: Медиасерверы, управление фотографиями, платформы для стриминга
- **Бизнес-приложения (Business Applications)**: CRM-системы, бухгалтерское ПО, платформы электронной коммерции (e-commerce)
- **Базы данных и аналитика (Databases & Analytics)**: Различные СУБД, инструменты мониторинга и аналитики
- **Коммуникация (Communication)**: Платформы для чата, почтовые серверы, форумы
- **И многое другое...**

#### Ключевые преимущества

- **Мгновенное развертывание**: Нет необходимости писать файлы Docker Compose с нуля
- **Автоматические обновления**: Сервисы можно обновлять одним кликом
- **Интегрированные функции**: Автоматические SSL-сертификаты и резервное копирование (backups)
- **Проверено сообществом**: Все сервисы тестируются и поддерживаются сообществом
- **Полный контроль**: Вы сохраняете полный контроль над своими данными и инфраструктурой

#### Добавление новых сервисов

Хотите добавить сервис в библиотеку Coolify? Мы приветствуем вклад (contributions) от сообщества!

Вы можете помочь путем:

- **Предложения новых сервисов**, которые будут полезны сообществу
- **Предоставления конфигураций Docker Compose** для приложений, которые вы используете
- **Улучшения существующих конфигураций сервисов** с помощью лучших настроек по умолчанию или функций
- **Написания документации**, чтобы помочь другим эффективно использовать сервисы

Узнайте, как добавлять новые сервисы, в нашем [руководстве по участию (contribution guide)](/get-started/contribute/service).
