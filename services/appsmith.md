# Appsmith

![Appsmith](https://raw.githubusercontent.com/appsmithorg/appsmith/release/static/images/appsmith-in-100-seconds.png)

## Что такое Appsmith

Организации создают внутренние приложения, такие как дашборды, GUI для баз данных, панели администратора, приложения для утверждений (approval apps), инструменты поддержки клиентов и т.д., чтобы помочь улучшить свои бизнес-операции.

Appsmith — это open-source инструмент для разработчиков, который позволяет быстро создавать такие приложения. Вы можете строить пользовательский интерфейс (UI) привычным способом drag-and-drop (перетаскиванием) из готовых виджетов.

Безопасно подключайтесь к своим базам данных и API с помощью встроенных источников данных (datasources). Пишите бизнес-логику для чтения и записи данных, используя запросы (SQL/API) и обычный JavaScript.

## Почему Appsmith

Appsmith позволяет легко создать UI, который общается с любым источником данных. Вы можете создать что угодно: от простых CRUD-приложений до сложных многоступенчатых рабочих процессов всего за несколько простых шагов:

- Подключение источника данных (Datasource): Интегрируйтесь с вашей базой данных или API. Appsmith поддерживает большинство популярных баз данных и REST/GraphQL API.
- Создание UI: Используйте встроенные виджеты для проектирования макета вашего приложения.
- Написание логики: Выражайте свою бизнес-логику с помощью запросов и JavaScript в любом месте редактора.
- Совместная работа, Деплой, Поделиться: Appsmith поддерживает контроль версий с использованием Git для совместной разработки приложений: используйте ветки для отслеживания и отката изменений. Разверните приложение и сделайте его доступным для других пользователей.
