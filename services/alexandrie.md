# Alexandrie

<ZoomableImage src="/docs/images/services/alexandrie.webp" alt="Интерфейс Alexandrie" />

## Что такое Alexandrie?

Alexandrie — это self-hosted приложение для ведения заметок в формате Markdown, созданное для скорости, работы офлайн и долгосрочного управления знаниями.  
Оно сфокусировано на скорости, дизайне, быстрой навигации по большим коллекциям заметок и явных ссылках между документами.

## Особенности

- Редактор Markdown с предпросмотром в реальном времени
- Быстрый полнотекстовый поиск, организация по тегам и иерархия
- Продвинутые функции управления для работы с сотнями документов
- Простое развертывание на базе Docker
