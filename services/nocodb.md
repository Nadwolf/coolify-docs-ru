# Что такое NocoDB?

<ZoomableImage src="/docs/images/services/nocodb1.webp" alt="Дашборд Nocodb" />

NocoDB — это open-source альтернатива Airtable. Превращает любую MySQL, PostgreSQL, SQL Server, SQLite и MariaDB в умную электронную таблицу (smart-spreadsheet).
