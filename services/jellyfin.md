## Что такое Jellyfin?

Jellyfin — это свободная система управления медиа (Free Software Media System), которая дает вам контроль над управлением и трансляцией (streaming) ваших медиафайлов. Это альтернатива проприетарным Emby и Plex, предназначенная для предоставления медиаконтента с выделенного сервера на устройства конечных пользователей через множество приложений. Jellyfin является ответвлением версии Emby 3.5.2 и портирован на фреймворк .NET Core для обеспечения полной кроссплатформенной поддержки.

Никаких подвохов, скрытых премиум-лицензий или платных функций: только команда, которая хочет создать что-то лучшее и работает вместе для достижения этой цели. Мы приветствуем всех, кто заинтересован присоединиться к нам в нашем стремлении!
