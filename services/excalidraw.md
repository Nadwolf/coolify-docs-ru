# Excalidraw

<ZoomableImage src="/docs/images/services/excalidraw-logo.svg" alt="Дашборд Excalidraw" />

## Что такое Excalidraw?

Excalidraw — это виртуальная доска (virtual whiteboard) для набросков диаграмм, выглядящих так, будто они нарисованы от руки. Это инструмент для совместного рисования, позволяющий создавать "ручные" схемы. Excalidraw идеально подходит для создания вайрфреймов (wireframing), мозговых штурмов, построения блок-схем (flowcharts) и сессий совместного дизайна. Приложение поддерживает совместную работу в реальном времени (real-time collaboration), экспорт в различные форматы и обладает интуитивно понятным интерфейсом, благодаря которому цифровое рисование (digital sketching) кажется естественным.
