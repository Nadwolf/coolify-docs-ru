## Что такое MetaMCP?

MetaMCP — это агрегатор/оркестратор MCP, который позволяет группировать несколько MCP-серверов в пространства имен (namespaces), применять промежуточное ПО (middleware) и предоставлять единую конечную точку (unified endpoint). Он поддерживает транспорты SSE и Streamable HTTP (плюс OpenAPI), работает с популярными клиентами MCP и создан для упрощения компоновки инструментов и их размещения за ключом API или доступом на основе OAuth.
