## Что такое Autobase?

**Autobase для PostgreSQL®** — это open-source альтернатива управляемым облачным базам данных (DBaaS), таким как Amazon RDS, Google Cloud SQL, Azure Database и другим.

Эта автоматизированная платформа баз данных позволяет вам создавать и управлять готовыми к продакшену высокодоступными кластерами PostgreSQL. Она упрощает процесс развертывания, снижает операционные расходы и делает управление базами данных доступным даже для команд без специализированного опыта.

*Легко автоматизируйте развертывание, обработку сбоев (failover), резервное копирование (backups), восстановление, обновления, масштабирование и многое другое.*

Попрощайтесь с ручным управлением базами данных 👋
