## Что такое Gotenberg?

Gotenberg предоставляет удобный для разработчиков API для взаимодействия с мощными инструментами, такими как Chromium и LibreOffice, для конвертации множества форматов документов (HTML, Markdown, Word, Excel и т.д.) в файлы PDF, и не только!

Gotenberg **не имеет графического интерфейса пользователя (UI)**. Перейдите к [документации](https://gotenberg.dev/docs/getting-started/introduction), чтобы узнать, как с ним взаимодействовать 🚀.
