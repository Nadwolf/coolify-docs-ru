# pgAdmin

<ZoomableImage src="/docs/images/services/pgadmin-logo.svg" />

## Что такое pgAdmin?

pgAdmin — это веб-инструмент (web-based) управления базами данных для администрирования ваших баз данных PostgreSQL через удобный интерфейс. Он предоставляет полный набор функций для администрирования баз данных, разработки запросов (query development) и визуализации данных, что делает его самой популярной и многофункциональной (feature-rich) платформой администрирования и разработки с открытым исходным кодом для PostgreSQL.
