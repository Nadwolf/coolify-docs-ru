# Что такое Apprise API?

[Apprise-api](https://github.com/caronc/apprise-api) использует всю мощь библиотеки [Apprise](https://github.com/caronc/apprise) в вашей сети через удобный API.

- Отправляйте уведомления более чем в 100 различных сервисов.
- Невероятно легкий шлюз (gateway) для работы с Apprise.
- Готовый к продакшену микросервис в вашем распоряжении.
- Простой веб-интерфейс для проверки и тестирования ваших конфигураций.

Apprise API был разработан для того, чтобы легко встраиваться в существующие (и новые) экосистемы, которым требуется простое решение для массовой рассылки уведомлений.
