## Что такое Logto?

Logto — это альтернатива Auth0, разработанная для современных приложений и SaaS-продуктов. Она предлагает бесшовный опыт для разработчиков (seamless developer experience) и хорошо подходит для частных лиц и растущих компаний.

🧑‍💻 **Комплексное решение для управления идентификацией от фронтенда до бэкенда**

- Позволяет осуществлять аутентификацию на основе OIDC с помощью Logto SDK.
- Поддерживает беспарольный вход (passwordless sign-in), а также различные варианты, такие как email, номер телефона, имя пользователя, Google, Facebook и другие методы социального входа (social sign-in).
- Предлагает красивые UI-компоненты с настраиваемым CSS в соответствии с потребностями вашего бизнеса.

📦 **Инфраструктура 'из коробки' (Out-of-the-box)**

- Включает в себя готовый к использованию Management API (API управления), служащий вашим провайдером аутентификации, что избавляет от необходимости дополнительной реализации.
- Предоставляет SDK, которые бесшовно интегрируют ваши приложения с Logto на различных платформах и языках программирования, адаптированные к вашей среде разработки.
- Предлагает гибкие коннекторы (connectors), которые могут масштабироваться за счет вклада сообщества и настраиваться с помощью протоколов SAML, OAuth и OIDC.

💻 **Решения корпоративного уровня (Enterprise-ready solutions)**

- Реализует управление доступом на основе ролей (role-based access control, RBAC) для масштабируемой авторизации ролей, обслуживая широкий спектр сценариев использования.
- Облегчает управление пользователями и предоставляет журналы аудита (audit logs) для понимания информации о пользователях, связанной с идентификацией, и обеспечения безопасности.
- Включает единый вход (single sign-on, SSO) и многофакторную аутентификацию (multi-factor authentication, MFA) без дополнительного написания кода.
- Использует Организации Logto (Logto Organizations) для легкого создания мультиарендных (multi-tenancy) приложений.

Более доступным языком мы называем это решение "[Управление доступом и идентификацией клиентов (Customer Identity Access Management, CIAM)](https://en.wikipedia.org/wiki/Customer_identity_access_management)" или просто "Решение для идентификации клиентов" (Customer Identity Solution).
