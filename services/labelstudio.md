# Что такое Label Studio?

<ZoomableImage src="/docs/images/services/labelstudio1.webp" alt="Дашборд Labelstudio" />
Label Studio — это open-source платформа для разметки данных (data labeling platform), которая оптимизирует процесс аннотирования и разметки данных для машинного обучения и приложений ИИ.
