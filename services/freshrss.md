# Freshrss

<ZoomableImage src="/docs/images/services/FreshRSS-logo.png" alt="Логотип Freshrss" />

## Что такое Freshrss

Бесплатный self-hosted агрегатор фидов (feed aggregator).

## Варианты развертывания

FreshRSS доступен в четырех конфигурациях развертывания в Coolify:

### FreshRSS (По умолчанию)
- **База данных:** SQLite (встроенная)
- **Сценарий использования:** Простые развертывания, тестирование или чтение персональных RSS-фидов
- **Компоненты:** Единый контейнер FreshRSS со встроенной базой данных SQLite

### FreshRSS с PostgreSQL
- **База данных:** PostgreSQL
- **Сценарий использования:** Prod-развертывания (production), требующие лучшей производительности и масштабируемости
- **Компоненты:**
  - Контейнер FreshRSS
  - Контейнер PostgreSQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### FreshRSS с MySQL
- **База данных:** MySQL
- **Сценарий использования:** Prod-развертывания с предпочтением MySQL
- **Компоненты:**
  - Контейнер FreshRSS
  - Контейнер MySQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### FreshRSS с MariaDB
- **База данных:** MariaDB
- **Сценарий использования:** Prod-развертывания с предпочтением MariaDB
- **Компоненты:**
  - Контейнер FreshRSS
  - Контейнер MariaDB
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)
