## Что такое Marimo?

Marimo — это open-source реактивный блокнот (reactive notebook) для Python: воспроизводимый, удобный для работы с git (git-friendly), со встроенным SQL, который можно выполнять как сценарий (script) и которым можно делиться (share) как приложением (app).
