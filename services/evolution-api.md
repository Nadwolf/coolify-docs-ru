## Что такое Evolution API?

Evolution API — это надежная платформа, созданная для расширения возможностей малого бизнеса, предпринимателей, фрилансеров и частных лиц с ограниченными ресурсами. Она предлагает гораздо больше, чем просто базовое решение для обмена сообщениями в WhatsApp™.

## Что нужно настроить для работы Coolify с Evolution API?

Доступный в Coolify docker-compose способен установить все необходимое для начала работы с Evolution API + Postgres + Redis + Evolution Manager.

Как только всё будет запущено, вы сможете получить доступ к https://url/manager.
