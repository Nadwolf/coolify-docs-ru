<ZoomableImage src="/docs/images/services/vvveb-logo.png" alt="Дашборд Vvveb" />

## Что такое Vvveb

Мощная и простая в использовании CMS для создания веб-сайтов, блогов или интернет-магазинов.

## Варианты развертывания

Vvveb доступен в трех конфигурациях развертывания в Coolify:

### Vvveb (По умолчанию)
- **База данных:** SQLite (встроенная)
- **Сценарий использования:** Простые развертывания, тестирование или небольшие веб-сайты
- **Компоненты:** Один контейнер Vvveb со встроенной базой данных SQLite

### Vvveb с MySQL
- **База данных:** MySQL
- **Сценарий использования:** Решения для продакшена с предпочтением MySQL
- **Компоненты:**
  - Контейнер Vvveb
  - Контейнер MySQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Vvveb с MariaDB
- **База данных:** MariaDB
- **Сценарий использования:** Решения для продакшена с предпочтением MariaDB
- **Компоненты:**
  - Контейнер Vvveb
  - Контейнер MariaDB
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)
