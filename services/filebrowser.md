## Что такое Filebrowser?

Filebrowser предоставляет интерфейс управления файлами (file managing interface) в пределах указанного каталога, с помощью которого можно загружать (upload), удалять, просматривать (preview), переименовывать и редактировать ваши файлы. Он позволяет создавать нескольких пользователей, и у каждого пользователя может быть свой собственный каталог (directory). Его можно использовать как самостоятельное (standalone) приложение.

## Настройка

- Разверните Filebrowser, используя шаблон Coolify
- Войдите в интерфейс (UI) Filebrowser с помощью следующих учетных данных (credentials):
  - Имя пользователя: `admin`
  - Пароль: сгенерирован случайным образом, его можно найти в логах (logs)
