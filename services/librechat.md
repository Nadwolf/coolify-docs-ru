<ZoomableImage src="/docs/images/services/librechat-logo.svg" alt="Дашборд Librechat" />

## Что такое LibreChat?

LibreChat — это мощный пользовательский интерфейс для чата (chat UI) для нескольких моделей ИИ, включая OpenAI GPT, Anthropic Claude, Azure OpenAI и многих других, созданный с фокусом на конфиденциальность (privacy-focused) для самостоятельного хостинга (self-hosted). Он предоставляет единый интерфейс для взаимодействия с различными ИИ-провайдерами, при этом сохраняя ваши диалоги приватными и под вашим контроля. LibreChat поддерживает ветвление диалогов (conversation threading), редактирование сообщений, интеграцию плагинов и настраиваемые пресеты (presets).
