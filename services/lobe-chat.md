# LobeChat

<ZoomableImage src="/docs/images/services/lobe-chat.screenshot.webp" />


## Что такое LobeChat?

LobeChat — это современный open-source фреймворк для ИИ-чата, созданный для всех. Он поддерживает работу с несколькими провайдерами ИИ, управление базой знаний и систему плагинов для обеспечения комплексного опыта общения с ИИ (AI chat experience).


## Возможности (Features)

- **Поддержка нескольких провайдеров (Multi-Provider Support)**: Совместимость с OpenAI, Claude, Gemini и многими другими ИИ-провайдерами.
- **База знаний (Knowledge Base)**: Встроенное управление базой знаний для улучшения ответов ИИ.
- **Система плагинов (Plugin System)**: Расширяемая архитектура плагинов для пользовательских функций (custom functionality).
- **Современный UI (Modern UI)**: Красивый и адаптивный пользовательский интерфейс.
- **Собственный хостинг (Self-Hosted)**: Полный контроль над вашими данными и конфиденциальностью.
- **Мультимодальность (Multi-Modal)**: Поддержка ввода текста, изображений и файлов.
