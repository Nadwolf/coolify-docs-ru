## Что такое Trigger?

Trigger — это фреймворк для фоновых заданий (Background Jobs) с открытым исходным кодом для TypeScript.

## Варианты развертывания

Trigger.dev доступен в двух конфигурациях развертывания в Coolify:

### Trigger.dev (По умолчанию)
- **База данных:** Встроенная PostgreSQL
- **Сценарий использования:** Стандартные развертывания с управляемой базой данных
- **Компоненты:**
  - Контейнер Trigger.dev
  - Встроенный контейнер PostgreSQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### Trigger.dev с внешней базой данных
- **База данных:** Внешняя (предоставляемая пользователем)
- **Сценарий использования:** Продвинутые развертывания с существующей инфраструктурой баз данных или пользовательскими конфигурациями БД
- **Компоненты:**
  - Контейнер Trigger.dev
  - Требуется переменная окружения `DATABASE_URL`, указывающая на вашу внешнюю базу данных
