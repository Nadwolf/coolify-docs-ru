# ESPHome

<ZoomableImage src="/docs/images/services/esphome-logo.svg" alt="Логотип ESPHome" />

## Что такое ESPHome?

ESPHome — это open-source фреймворк прошивок (firmware framework), который упрощает процесс создания кастомных прошивок для популярных микроконтроллеров с поддержкой WiFi.
