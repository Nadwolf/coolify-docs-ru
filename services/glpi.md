## Что такое GLPI

GLPI (Gestionnaire Libre de Parc Informatique) — это бесплатная open-source платформа управления ИТ-услугами (IT Service Management, ITSM), используемая для управления ИТ-активами, службы поддержки (helpdesk) и операций service desk, помогающая организациям проводить инвентаризацию оборудования/программного обеспечения, управлять инцидентами, отслеживать запросы и оптимизировать ИТ-поддержку с помощью таких функций, как CMDB, система тикетов и автоматизация.

### Начальная настройка

После развертывания шаблона GLPI автоматически завершит процесс установки. Пожалуйста, подождите около 4–5 минут до полного запуска. Затем вы можете войти в систему с помощью:

   ```bash
   # User
   glpi

   # password
   glpi
   ```
