# Supertokens

<ZoomableImage src="/docs/images/services/supertokens.png" alt="Дашборд Supertokens" />

## Что такое Supertokens

Решение для аутентификации с открытым исходным кодом, которое упрощает реализацию безопасной аутентификации пользователей и управления сессиями для веб- и мобильных приложений.

## Варианты развертывания

SuperTokens доступен в двух конфигурациях развертывания в Coolify:

### SuperTokens с MySQL
- **База данных:** MySQL
- **Сценарий использования:** Развертывания в продакшене с предпочтением MySQL
- **Компоненты:**
  - Контейнер SuperTokens
  - Контейнер MySQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

### SuperTokens с PostgreSQL
- **База данных:** PostgreSQL
- **Сценарий использования:** Развертывания в продакшене с предпочтением PostgreSQL
- **Компоненты:**
  - Контейнер SuperTokens
  - Контейнер PostgreSQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

Оба варианта обеспечивают эквивалентную функциональность — выбирайте тот, который соответствует вашим предпочтениям в отношении базы данных или существующей инфраструктуре.
