## Что такое Code Server?

Запускайте [VS Code](https://github.com/Microsoft/vscode) на любой машине где угодно и получайте к нему доступ через браузер.

## Основные преимущества (Highlights)

- Пишите код на любом устройстве в единой среде разработки (consistent development environment)
- Используйте облачные серверы для ускорения тестов, компиляции, скачиваний и многого другого
- Экономьте заряд батареи, когда вы в пути: все ресурсоемкие задачи выполняются на вашем сервере

## Требования

Смотрите [требования](https://coder.com/docs/code-server/latest/requirements?utm_source=coolify.io) для минимальных характеристик.

**Коротко говоря:** Linux-машина с включенными WebSockets, 1 ГБ ОЗУ (RAM) и 2 vCPU
