# Redmine

<ZoomableImage src="/docs/images/services/redmine-logo.svg" alt="Логотип Redmine" />

## Что такое Redmine?

Redmine — это гибкое веб-приложение для управления проектами. Написанное с использованием фреймворка Ruby on Rails, оно является кроссплатформенным и поддерживает различные базы данных.
