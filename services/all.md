# Все сервисы

Полный каталог всех сервисов в один клик, доступных в Coolify, организованный по категориям.

## Администрирование

- [Cockpit](/services/cockpit) — Веб-интерфейс для администрирования серверов.
- [Dashboard](/services/dashboard) — Простая панель управления для вашего сервера.
- [Dashy](/services/dashy) — Настраиваемая домашняя страница (дашборд) для self-hosted сервисов.
- [Glance](/services/glance) — Универсальный дашборд для домашнего сервера.
- [Heimdall](/services/heimdall) — Элегантное решение для организации всех ваших веб-приложений.
- [Homarr](/services/homarr) — Настраиваемая домашняя страница и дашборд для браузера.
- [Homepage](/services/homepage) — Современная домашняя страница для вашего сервера.
- [LiteQueen](/services/litequeen) — Легковесная платформа для управления сервисами.
- [Organizr](/services/organizr) — Органайзер домашних страниц для сервисов вашего сервера.

## ИИ (Искусственный интеллект)

- [AnythingLLM](/services/anythingllm) — OpenAI-совместимый LLM-клиент, позволяющий разработчикам быстро создавать и масштабировать рабочие процессы.
- [MetaMCP](/services/metamcp) — Агрегатор MCP, оркестратор, промежуточное ПО и шлюз в одном приложении.
- [Argilla](/services/argilla) — Платформа с открытым исходным кодом для создания, обучения и оценки моделей разговорного ИИ.
- [Chroma](/services/chroma) — Нативная для ИИ векторная база данных для создания приложений с эмбеддингами.
- [Flowise](/services/flowise) — Визуальный интерфейс (drag & drop) для создания собственных цепочек (flows) LLM.
- [Label Studio](/services/labelstudio) — Платформа с открытым исходным кодом для разметки данных.
- [Langflow](/services/langflow) — Инструмент на базе Python для создания ИИ-агентов и рабочих процессов с визуальным интерфейсом.
- [Langfuse](/services/langfuse) — Платформа для инженерии и мониторинга LLM приложений.
- [LibreChat](/services/librechat) — Мощный и ориентированный на конфиденциальность чат-интерфейс для нескольких моделей ИИ.
- [LibreTranslate](/services/libretranslate) — Бесплатный API машинного перевода с открытым исходным кодом.
- [LobeChat](/services/lobe-chat) — Современный фреймворк для ИИ-чатов с поддержкой нескольких провайдеров и управлением базой знаний.
- [LiteLLM](/services/litellm) — Шлюз LLM для управления аутентификацией, балансировкой нагрузки и отслеживанием расходов в 100+ моделях LLM.
- [MindsDB](/services/mindsdb) — Платформа машинного обучения, которая привносит ИИ в базы данных.
- [NewAPI](/services/newapi) — Шлюз LLM следующего поколения и система управления ИИ-активами.
- [Ollama](/services/ollama) — Легковесный и эффективный сервер для запуска больших языковых моделей (LLM) локально или в облаке.
- [Open WebUI](/services/open-webui) — Удобный веб-интерфейс для LLM (ранее Ollama WebUI).
- [OpenClaw](/services/openclaw) — ИИ-помощник для написания кода с поддержкой нескольких провайдеров и автоматизацией браузера.
- [Qdrant](/services/qdrant) — Векторная база данных с открытым исходным кодом, ориентированная на ИИ.
- [Unstructured](/services/unstructured) — Инструменты для извлечения и обработки неструктурированных документов для систем RAG и тонкой настройки моделей.
- [Weaviate](/services/weaviate) — Нативная для ИИ векторная база данных с открытым исходным кодом.
