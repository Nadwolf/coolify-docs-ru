# Что такое Calibre Web Automated with Downloader

## Calibre Web Automated

Calibre, несмотря на то что это фантастический для своего времени инструмент, имеет ряд проблем при контейнеризации (containerised). К ним относится зависимость от инстанса сервера KasmVNC для отображения UI, который практически невозможно использовать на мобильных устройствах и который довольно требователен к ресурсам, если у вас небольшой маломощный сервер (как у меня).

### Ссылки

## Downloader

Интуитивно понятный веб-интерфейс для поиска и запроса на скачивание (requesting) книг, созданный для бесшовной работы с Calibre-Web-Automated. Этот проект упрощает процесс скачивания книг и их подготовки к интеграции в вашу библиотеку Calibre.

### Ссылки

- [GitHub](https://github.com/calibrain/calibre-web-automated-book-downloader)
