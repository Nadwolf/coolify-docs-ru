# Linkding

<ZoomableImage src="/docs/images/services/linkding-logo.svg" alt="Логотип Linkding" />

## Что такое Linkding?

Linkding — это минималистичный self-hosted менеджер закладок (bookmark manager), который фокусируется на простоте и производительности. Он предоставляет чистый интерфейс для сохранения, организации и поиска ваших закладок.

## Различные версии Linkding

Linkding доступен в двух версиях:

| Версия          | Описание |
|-----------------|-------------|
| `latest`        | Предоставляет базовую функциональность linkding |
| `plus`          | Включает функцию для архивирования веб-сайтов в виде снимков HTML (HTML snapshots). Примечание: эта версия имеет значительно больший размер образа (image size), так как включает установку Chromium, требует больше оперативной памяти (runtime memory) для запуска Chromium и требует больше дискового пространства для хранения снимков HTML. |
