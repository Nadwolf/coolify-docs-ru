# Langflow

<ZoomableImage src="/docs/images/services/langflow.svg" alt="Langflow" />

## Что такое Langflow?

Langflow — это open-source фреймворк на базе Python для создания ИИ-приложений. Он объединяет возможности визуального создания (visual authoring) с полным программным доступом, позволяя командам разрабатывать прототипы, тестировать и развертывать ИИ-агентов (AI-powered agents) и рабочие процессы.

Он ускоряет разработку, предоставляя **интерфейс drag-and-drop**, сохраняя при этом гибкость **кастомных компонентов Python** для продвинутых сценариев использования.

## Учетные данные (Credentials)

После развертывания Langflow в Coolify вы можете войти в дашборд с автоматически сгенерированными учетными данными, которые хранятся в следующих переменных окружения:

| Переменная                     | Описание                         |
|--------------------------------|----------------------------------|
| `LANGFLOW_SUPERUSER`           | Имя пользователя (username) для аккаунта администратора |
| `LANGFLOW_SUPERUSER_PASSWORD`  | Пароль (password) для аккаунта администратора |
