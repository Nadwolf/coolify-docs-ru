## Что такое BudgE?

BudgE (произносится как «budgie», в честь птички (волнистый попугайчик)) — это приложение с открытым исходным кодом (open source) для управления личными финансами методом «бюджетирования по конвертам» (budgeting with envelopes). На его создание вдохновили такие инструменты, как [Aspire Budgeting](https://www.aspirebudget.com?utm_source=coolify.io), [budgetzero](https://budgetzero.io?utm_source=coolify.io) и [Buckets](https://www.budgetwithbuckets.com/?utm_source=coolify.io).

## Текущие возможности

- Поддержка нескольких пользователей
- Бюджетирование по конвертам (Envelope budgeting) с возможностью переноса остатка на следующий месяц
- Управление транзакциями по счетам
- Управление стандартными банковскими счетами
- Управление кредитными картами с обработкой платежей
- Отслеживаемые (Tracking) счета
- Экспорт транзакций по счету
- Импорт транзакций из CSV
