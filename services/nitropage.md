## Что такое Nitropage?

Nitropage — это расширяемый визуальный конструктор веб-сайтов на базе SolidStart, предлагающий растущую библиотеку универсальных строительных блоков (building blocks).

## Варианты развертывания

Nitropage доступен в двух конфигурациях развертывания в Coolify:

### Nitropage (По умолчанию)
- **База данных:** SQLite (встроенная)
- **Сценарий использования:** Простые развертывания, тестирование или небольшие веб-сайты
- **Компоненты:** Единый контейнер Nitropage со встроенной базой данных SQLite

### Nitropage с PostgreSQL
- **База данных:** PostgreSQL
- **Сценарий использования:** Prod-развертывания (production), требующие лучшей производительности и масштабируемости
- **Компоненты:**
  - Контейнер Nitropage
  - Контейнер PostgreSQL
  - Автоматическая настройка базы данных и проверки работоспособности (health checks)

## Возможности (Features)

- Переиспользуемые **Пресеты (Presets)** элементов и **Макеты (Layouts)**
- Рабочий процесс публикации с **Ревизиями (Revisions)**
- Стартовый набор с десятками чертежей (Blueprints)
- Несколько веб-сайтов на одном экземпляре
- Оптимизация изображений с кадрированием по **фокусной точке (focal point)**
- API для ваших кастомных элементов страницы [SolidJS](https://www.solidjs.com/?utm_source=coolify.io)
- Мощная архитектура для разработчиков с [Vite](https://vitejs.dev/?utm_source=coolify.io)
- Автоматическая обработка sitemap.xml и Atom-лент
