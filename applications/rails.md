# Ruby on Rails

Ruby on Rails — это фреймворк для создания веб-приложений, который включает в себя всё необходимое для разработки приложений на основе баз данных в соответствии с паттерном Model-View-Controller (MVC).

## Требования

Если вы хотите выполнять миграцию базы данных во время развертывания с использованием билдпака `NIXPACKS`, вам необходимо задать следующую команду запуска (`Start Command`):

```bash
bundle exec rake db:migrate && bundle exec bin/rails server -b 0.0.0.0 -p ${PORT:-3000} -e $RAILS_ENV
```
