# Интеграция с GitHub
Coolify упрощает развертывание приложений из ваших репозиториев GitHub или Docker-образов, размещенных в GitHub Container Registry.

Интеграция GitHub с Coolify поддерживает деплой как из приватных, так и из публичных репозиториев, автоматические развертывания при новых коммитах и развертывания для Pull Request.


## Способы использования GitHub с Coolify

Вы можете интегрировать GitHub с Coolify несколькими способами, в зависимости от ваших потребностей. Ниже приведены доступные варианты, каждый из которых ссылается на подробное руководство для легкой настройки:

| Метод | Описание |
|--------|-------------|
| [Публичный репозиторий](/applications/ci-cd/github/public-repository) | Развертывайте приложения напрямую, используя URL публичного репозитория. |
| [Приватный репозиторий через GitHub App](/applications/ci-cd/github/setup-app) | Установите GitHub App в свой личный аккаунт или организацию для деплоя как приватных, так и публичных репозиториев. |
| [Приватный репозиторий через Deploy Key](/applications/ci-cd/github/deploy-key) | Развертывайте приложения из приватных репозиториев, используя ключ развертывания (deploy key). |
| [Автоматические развертывания](/applications/ci-cd/github/auto-deploy) | Автоматически развертывайте новые версии приложения при пуше коммитов в определенную ветку вашего репозитория GitHub. |
| [Сборка и деплой через GitHub Actions](/applications/ci-cd/github/actions) | Собирайте приложение на GitHub с помощью GitHub Actions как часть вашего CI/CD конвейера, пушьте его в любой Docker-реестр (например, GHCR или Docker Hub) и автоматически развертывайте в Coolify. |
| [Предварительные развертывания](/applications/ci-cd/github/preview-deploy) | Автоматически развертывайте новые версии вашего приложения на основе Pull Request. |
