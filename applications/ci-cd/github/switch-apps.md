# Смена GitHub Apps
Смена GitHub Apps позволяет изменить приложение GitHub, связанное с вашими развернутыми приложениями, например, при переносе репозиториев в новую организацию или аккаунт.


### Зачем переключать GitHub Apps?
1. **Изменения в организации**: вы перенесли репозитории в новую организацию GitHub, которая требует другого приложения GitHub App.
2. **Управление доступом**: вы хотите использовать GitHub App с другими правами доступа или доступом к репозиториям.


::: warning ВАЖНО
Эта функция представлена в **Coolify v4.0.0-beta.408**. Чтобы следовать этому руководству, вы **должны** использовать Coolify версии v4.0.0-beta.408 или выше.
:::

## 1. Перенос репозиториев (Опционально)
Если ваша цель — перенести репозиторий в другой аккаунт или организацию, сначала выполните трансфер на стороне GitHub.

Если вы просто хотите использовать другое GitHub App без смены репозиториев, вы можете пропустить этот шаг.

## 2. Добавьте новое GitHub App в Coolify
У нас есть отдельное руководство по настройке GitHub App: [/applications/ci-cd/github/setup-app](/applications/ci-cd/github/setup-app). Следуйте ему, а затем возвращайтесь сюда.

## 3. Настройте доступ к репозиториям
После добавления нового GitHub App убедитесь, что оно имеет доступ к нужным репозиториям:

<ZoomableImage src="/docs/images/applications/ci-cd/github/switch-apps/1.webp" />
1. В дашборде Coolify нажмите **Sources** в боковом меню и выберите ваше новое приложение GitHub App.
2. Нажмите кнопку **Update Repositories** (это перенаправит вас на GitHub).

<ZoomableImage src="/docs/images/applications/ci-cd/github/switch-apps/2.webp" />
3. В разделе «Repository access» проверьте, видите ли вы репозитории, которые хотите использовать.

## 4. Смените Git Source в Coolify
<ZoomableImage src="/docs/images/applications/ci-cd/github/switch-apps/3.webp" />
1. Откройте страницу конфигурации вашего приложения.
2. Перейдите на страницу «Git Source».
3. Выберите ваше новое приложение GitHub App.

## 5. Обновите название репозитория (если применимо)
<ZoomableImage src="/docs/images/applications/ci-cd/github/switch-apps/4.webp" />
Если вы перенесли репозиторий в новую организацию или аккаунт, обновите название репозитория соответствующим образом (например, с `shadowarcanist/switch-apps-guide` на `airoflare/switch-apps-guide`).

Вы можете пропустить этот шаг, если не переносили репозиторий.

## 6. Переразверните приложение
Чтобы применить новое GitHub App к вашему приложению, запустите процесс переразвертывания, нажав кнопку **Redeploy**.
