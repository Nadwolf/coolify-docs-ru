# VitePress

Генератор статических сайтов на базе Vite и Vue. Превращайте Markdown в красивую документацию за считанные минуты.

[Пример репозитория.](https://github.com/coollabsio/coolify-examples/tree/main/vitepress)

## Сборка (статическая)

- Используйте `Nixpacks`.
- Включите опцию `Is it a static site?`.
- Установите `Base Directory` (базовую директорию) равной `/.vitepress/static`.
- Установите `Publish Directory` (публичную директорию) равной `/.vitepress/dist`.
