# Vue

Vue.js — это доступный, производительный и универсальный фреймворк для создания веб-интерфейсов.

[Пример репозитория.](https://github.com/coollabsio/coolify-examples/tree/main/vue)

## Серверная сборка (NodeJS|Express)

- Установите `Build Pack` в значение `nixpacks`.
- Установите команду запуска (`Start Command`) в значение `node server.js`.

## Статическая сборка (SPA)

- Установите `Build Pack` в значение `nixpacks`.
- Включите опцию `Is it a static site?`.
- Установите `Output Directory` (исходящая директория) в значение `dist`.

## Статическая сборка с Роутером (SPA)

- Установите `Build Pack` в значение `nixpacks`.
- Включите опцию `Is it a static site?`.
- Установите `Output Directory` (исходящая директория) в значение `dist`.
