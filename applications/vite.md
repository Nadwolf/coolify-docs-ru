# Vite

Vite — это инструмент сборки, цель которого — обеспечить более быстрый и легкий опыт разработки для современных веб-проектов.

[Пример репозитория.](https://github.com/coollabsio/coolify-examples/tree/main/vite)

## Сборка Vanilla TypeScript (статическая)

- Установите `Build Pack` в значение `nixpacks`.
- Включите опцию `Is it a static site?`.
- Установите `Publish Directory` (публичную директорию) равной `dist`.


## Сборка Vanilla JavaScript (статическая)

- Установите `Build Pack` в значение `nixpacks`.
- Включите опцию `Is it a static site?`.
- Установите `Publish Directory` (публичную директорию) равной `dist`.
