# Phoenix

Phoenix — это производительный веб-фреймворк на языке Elixir/Erlang, который не идет на компромисс в скорости и поддерживаемости.

## Требования

- Установите `Build Pack` в значение `nixpacks`.
- Установите переменную `MIX_ENV` в значение `prod`.
  - Это должна быть переменная окружения времени сборки (`Build time`).
- Установите `SECRET_KEY_BASE` в виде случайной строки ([документация Phoenix по деплою](https://hexdocs.pm/phoenix/deployment.html#handling-of-your-application-secrets)).
  - Это должна быть переменная окружения времени сборки (`Build time`).
- Установите `DATABASE_URL` — строку подключения к вашей базе данных.
  - Это должна быть переменная окружения времени сборки (`Build time`).
- Установите `Ports Exposes` (проброс портов) в значение `4000` (по умолчанию).
